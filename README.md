2DGameKurs
==========

1.
http://windows.github.com/
Installieren

2. 
Github Account anlegen

3.
https://github.com/ADoby/2DGameKurs
Link folgen und rechts auf "Clone in Windows" klicken

4. 
Github sollte sich öffnen und anfangen die SVN runterzuladen

5.
Nach "Dokumente\GitHub" navigieren 
und den ordner "2DGameKurs" in den "projects" ordner von "Cocos2d-x Beta 3"
verschieben

Ordner Struktur dann
"Cocos2d.x Beta3\projects\2DGameKurs\proj.win32\2DGame.sln"

6.
Wenn ihr jetzt bei GitHub nochmal versucht doppelt auf 2DGameKurs zu klicken sagt er Fehler
da klickt auf "Find It" und wählt den Ordner "2DGameKurs" im "projects" Ordner aus.

Nun ist alles richtig eingestellt.

In GitHub könnt ihr nun immer schauen ob es eine neue Version gibt und sie updaten lassen (sync oben)
